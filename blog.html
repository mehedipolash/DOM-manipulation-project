Q2: What is the difference between innerHTML, innerText, and textContent?

1. innerHTML: it reads both the HTML markup and the text content of the element
 2. innerText: it focuses on the rendered text content. it returns the text as it appears on screen.
 3. textContent: it also ignores all HTML tags and returns only the text.it only deals with the raw text and doesn't account for styles.
